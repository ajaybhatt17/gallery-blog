# gallery-blog

Based on other repo - https://github.com/contentful/gallery-app-react/

## Install

`yarn install`
or
`npm install`

`cp config.example.js config.js`

replace deliveryAccessToken with dropbox access token


## Deploy

`npm run build`
or
`yarn run build`

`npm run start-server`
or
`yarn run start-server`
